🌦️ Weather Application

A simple and responsive Weather App built using HTML, CSS, and JavaScript that shows real-time weather information for selected cities.

📌 Features

🌍 Get current weather details of different cities

🌡️ Displays Temperature, Humidity, and Weather Condition

💨 Shows Wind Speed

📱 Fully Responsive Design (Works on Mobile + Desktop)

🔽 City selection using Dropdown

⚡ Fast and Lightweight (No frameworks used)

🛠️ Built With

HTML → Structure of the application

CSS → Styling and Responsive Design

JavaScript → Logic and Weather Data Handling

📷 Preview

Shows:

City Name & Country

Temperature in °C

Weather Condition (Clear, Clouds, Rain, etc.)

Humidity Level

Wind Speed
